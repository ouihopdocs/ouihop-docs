publishPedestrianDestinationWithCoordinate:address:zipcode:city:activeAfter:

- (void)publishPedestrianDestinationWithCoordinate:(CLLocationCoordinate2D)coordinate 
        address:(NSString*)address zipcode:(NSString*)zipcode city:(NSString*)city 
        activeAfter:(NSNumber*)activeAfter;
Parameters
coordinate

Geo-coordinate of the destination.

address

Reversely geocoded address of the destination.

zipcode

Zipcode of the destination.

city

City of the destination.

activeAfter

An NSNumber holding the number of minutes to publish the destination after. This should be a non-negative integer. Both nil or zero values will immediately publish the destination.

Discussion

Publishes a new pedestrian destination to drivers.
