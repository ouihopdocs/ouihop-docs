cancelJoinRequestForced:

- (void)cancelJoinRequestForced:(BOOL)shouldForceCancellation;
Parameters
shouldForceCancellation

If this variable is YES, current trip request will be discarded locally, even if failed to cancel it at server side.

Discussion

Cancels a previously created join request for a trip.

cancelWithDisputeAcceptedRequestForced:

- (void)cancelWithDisputeAcceptedRequestForced:(BOOL)shouldForceCancellation;
Parameters
shouldForceCancellation

If this variable is YES, current trip request will be discarded locally, even if failed to cancel it at server side.

Discussion

Cancels a previously accepted request for a trip. This method should be used when a driver incorrectly signals this request as BOARDED while a pedestrian isn't really boarded. A record is kept for this request that boarding was disputed by pedestrian.

dropOffRequest

- (void)dropOffRequest;
Discussion

Specifies that a pedestrian was dropped off the car for a boarded request.
