startListeningToPedestrianDestinations:

- (void)startListeningToPedestrianDestinations:(NSObject<PedestrianDestinationRequestInteractorObserver>*_Nonnull)observer;
Parameters
observer

An observer to be attached, that implements needed protocol. If this observer already exists, nothing happens.

Discussion

* Attaches an observer to the list of interactor observers. If this is the first observer, interactor starts listening for surrounding pedestrian destinations in real-time manner. If an observer is already attached to the interactor, only the new observer is attached.

*

stopListeningToPedestrianDestinations:

- (void)stopListeningToPedestrianDestinations:(NSObject<PedestrianDestinationRequestInteractorObserver>*_Nonnull)observer;
Parameters
observer

An observer to be detached. If this observer doesn't exist, nothing happens.

Discussion

* Detaches an observer from the list of interactor observers. If this is the last observer, interactor stops listening for surrounding pedestrian destinations real-time updates. If other observers are still attached to the interactor, the new observer is only attached.

*
