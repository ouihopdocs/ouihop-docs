LocationInteractorObserver

@protocol LocationInteractorObserver <NSObject>

A protocol that can be implemented by any object that is interested in location change events.

  • -didUpdateToLocation:

    A method called for all location observers when location is updated. Location is considered updated if an accurate location is acquired and its different from last acquired location.

    Declaration

    Objective-C

    - (void)didUpdateToLocation:(id)location;

    Parameters

    location

    The new acquired location. This location can be later retrieved from currentLocation property.
  • -didUserMoveForDistance:

    An optional method to notify observers with distance difference between last and current locations.

    Declaration

    Objective-C

    - (void)didUserMoveForDistance:(id)distance;

    Parameters

    distance

    Distance in meters between last and current locations.
