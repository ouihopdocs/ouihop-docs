  • SurroundingTripsInteractor

    An interactor responsible for retrieving surrounding trips that match needed criteria, depending on current state.

    Note

    This interactor makes use of two SurroundingUsersInteractor’s. One that should already exist in the SDK with a short interval time and a quite narrow Geohash filter. Another internal SurroundingUsersInteractor instance is created with a long interval time, but a wider Geohash filter. Both interactors are used to update drivers’ locations for surrounding trips.

    Also a PedestrianDestinationInteractor is being used to determine trips filtration using current pedestrian destination if exists.

    See more

    Declaration

    Objective-C

    @interface SurroundingTripsInteractor
    : NSObject <SurroundingUsersInteractorObserver, LocationInteractorObserver,
                PedestrianDestinationInteractorObserver>
